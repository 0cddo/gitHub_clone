다양한 미디어 유형이나 장치에 따라, 서로 다른 스타일 규칙 적용 -> @media 미디어타입 and (미디어특성) { css 코드 } -> 미디어 타입 : all(모든) / screen(모니터) / print(인쇄 전용) / 생략가능 -> 미디어 특성 : width / max-width(~값 이하) / min-width(~값 이상) / height... / orientation (portrait, landscape / 핸드폰 방향에 따른 스타일 적용) -> 참고) https://getbootstrap.com/docs/5.0/layout/grid/
